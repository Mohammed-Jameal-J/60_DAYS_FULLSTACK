# Shopping App Project (Single Store)

## Project Goal

Build a simple **customer-facing shopping application for a single store** where users can browse products, add items to cart, place orders, and receive home delivery from that store.

## Project Description

This project is a **single-store eCommerce application** focused only on the **user/customer side of the platform**. Customers can create an account, browse products available in the store, add items to their cart, and place orders for delivery. The application will provide a smooth shopping experience similar to modern online stores but simplified for a single shop.

The current scope focuses only on the **User App (Customer Side)**. Admin and delivery systems can be added later.

---

# User App Features

## 1. User Authentication

* User signup
* User login
* Login with email and password
* Forgot password / Reset password
* Google login (optional)
* OTP login (optional)
* Logout

## 2. User Profile

* View profile
* Edit profile information
* Save phone number
* Manage delivery addresses
* Add multiple addresses
* Select default delivery address

## 3. Home Page

* Featured products
* New arrivals
* Popular products
* Product categories
* Search bar

## 4. Product Browsing

* View all products
* Browse by category
* Product search
* Product filtering

  * Price
  * Category
  * Availability

## 5. Product Details Page

Each product page includes:

* Product images
* Product name
* Product description
* Product price
* Available stock
* Add to cart button

## 6. Shopping Cart

* Add product to cart
* Increase or decrease quantity
* Remove product from cart
* View cart summary
* View total price

## 7. Checkout Process

* Select delivery address
* Review cart items
* View total price
* Select payment method

Payment methods may include:

* Cash on Delivery (COD)

* Online payment via Razorpay

* Place order button

## 8. Order Management (User Side)

Users can:

* View order history
* View order details
* Track order status

Example order statuses:

* Order placed
* Order confirmed
* Packed
* Out for delivery
* Delivered

## 9. Notifications

* Welcome email sent after user signup
* Email notification for order placed confirmation
* Email notification for order status updates
* Email notification for delivery updates
